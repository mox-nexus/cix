# cix

Cognitive extensions that amplify human capability, not replace it.

## Install

**Claude Code marketplace**
```bash
claude marketplace add mox-labs/cix
```

**CLI tool**
```bash
uvx --from "cix @ git+https://github.com/mox-nexus/cix#subdirectory=tools/cix" cix
```
