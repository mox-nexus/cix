# cix

Cognitive extensions that amplify human capability, not replace it.

## Install

**CLI tool**
```bash
uv tool install git+https://github.com/mox-labs/cix
```

**Claude Code plugin**
```
/plugin install https://github.com/mox-labs/cix
```
