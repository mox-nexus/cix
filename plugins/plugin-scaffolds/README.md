# plugin-scaffolds

CI methodology for building cognitive extensions. Composes with `plugin-dev`.

## When to Use

- Creating extensions that enhance human capability
- Applying transparency/control patterns
- Setting up dual-content structure (Claude vs Human)
- Designing agents with Orthogonality Locks

## Composes With

- **plugin-dev**: Structure, schemas, frontmatter, hooks API
- **This plugin**: CI methodology, dual-content, transparency, control plane

## Skills

- **ci-extension-design**: Methodology for collaborative intelligence extensions
