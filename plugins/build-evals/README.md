# build-evals

Eval methodology for AI systems. Write evals that measure what matters.

## When to Use

- Building test suites for agents, skills, MCP servers
- Measuring agent effectiveness
- Evaluating multi-agent coordination
- Choosing eval frameworks

## Skills

- **build-eval**: Eval methodology covering grader types, metrics, frameworks

## Key Concepts

- **Three grader types**: Code-based, Model-based, Human
- **Non-determinism**: pass@k, pass^k for stochastic outputs
- **Match eval to agent type**: Coding, Conversational, Research, Computer Use, Multi-Agent
