# CI Lab Naming: CEP = Collaboration Enhancement Proposal

**Date**: 2025-02-28
**Participants**: user, claude

user: The experiments in ci-lab need a name. Something like RFC but for collaborative intelligence experiments.

claude: Options:
1. **CIE** — Collaborative Intelligence Experiment
2. **CEP** — Collaboration Enhancement Proposal
3. **CIP** — Collaborative Intelligence Proposal

user: CEP. Collaboration Enhancement Proposal. It covers extensions, plugin design, agent design, interaction strategies, documentation formats, prompt framing.

claude: CEP it is. Current CEPs:
- **CEP-001**: Mandate framing (WHY vs HOW in system prompts)
- **CEP-002**: Mandate framing expanded (completed)
- **CEP-003**: Skills vs references — do skill-format docs improve agent performance over mechanical --help?

user: CEP-003 is about what skills can do that mechanical references cannot. Not about proving skills are redundant — about discovering their unique value.

**Decision**: CEP = Collaboration Enhancement Proposal. Covers all experiment types in the ci-lab.

**Key fact**: CEP-003 has three subjects: help-only, skill-at-tool, help-plus-agent-skill.
